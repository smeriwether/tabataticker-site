FAQ

Can I customize the workout?

Not currently. Tabata Ticker is intentionally fixed to the classic 8-round, 20-second work, 10-second rest format.

Does the Watch run the timer by itself?

The iPhone owns the timer. Apple Watch mirrors the current state and sends control commands back to the phone.
